リリース情報

機械猫モッカーのリリース履歴です。

1.0.1 Release - 19 Dec 2006

バグ・フィックス

  • オブジェクトのキャストに失敗する

1.0 Release - 2 Dec 2006

追加機能

  • 制御構文(if, for, while, do-while)に対応
  • フィールド定義の許可
  • assertの緩和

仕様変更

  • 不定値の導入
  • 擬似クラスの挙動の拡張
    「Beyond the Test」の考えを導入
    • 副作用メソッドの判定
    • ヒストリーの変更

1.0 Release (Alpha) - 7 Sept 2006

追加機能

  • 全プリミティブ型に対応
  • setUp() メソッドのオーバーライド必要性の排除
  • tearDown() メソッドのオーバーライド禁止の排除
  • キャスト対応
  • デフォルト・パッケージ対応
  • 例外テスト・ケース対応
  • ツールのエラー・ハンドリング強化
    エラー・メッセージによる原因の明確化
  • 擬似クラスのDIやファクトリー・メソッドによる指定

仕様変更

  • 擬似クラスの挙動拡張
    「Test as Definition」から「Test as Scenario」の考えを導入

既知の問題

  • プライグインからの実行を行った際に、UnsupportedClassVersionError が発生する場合がある

0.9.6 Release - 26 July 2006

追加機能

  • 任意のクラス・ロード機能

仕様変更

  • オブジェクトの保存形式の変更
